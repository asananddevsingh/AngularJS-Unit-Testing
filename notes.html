Steps to setup project.

Open cmd at "build" folder level and configure package.json file using command: npm init

Install dependencies:
npm install karma --save-dev
npm install karma-jasmine --save-dev
npm install karma-chrome-launcher --save-dev

Initialize karma configuration: karma init

Configure it with: testing framework: jasmine
Require.js: No
browser: chrome
location: 1) app/**/*.js. 2) test/**/*.js.
Everything else as defaults.
